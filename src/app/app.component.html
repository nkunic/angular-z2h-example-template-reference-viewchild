Template (Local) Reference Variable & @ViewChild decorator

  • Can be added to any element in DOM using # as the first character
  • Using Template reference we can access DOM elements and their properties
  • Can be used only in the HTML template
  • To access in TS component we need to explicitly pass the reference to TS trough an event in the HTML template.
  • Sometimes we need to get access to the DOM before we trigger an event in HTML. For that type of usecase, we can use @ViewChild
